# Clase 1: Proyecto Demo - Flutter Explorer UX/UI

# e1_widghets_clase1
Se comparte un proyecto con la finalidad de explorar la estructura de los widghets, tipos de touch y algo de navegacion 
## 🎯 Objetivo de la clase

Construir una app móvil funcional en Flutter que sirva como **caso de estudio vivo** para entender:
- Widgets `Stateless` y `Stateful`
- Árbol de widgets
- Layout (Column, Row, Stack)
- Espaciado y tamaño
- Estilo visual y tema global
- Interacción y navegación
- Principios UX/UI (Jerarquía visual, Consistencia, Feedback, Affordance)

---

## 📱 Nombre del proyecto

**Flutter Explorer: UX/UI en acción**
